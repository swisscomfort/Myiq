# DSGVO / GDPR Hinweise (Kompakt)

Wichtige Pflichten:
- Einwilligung (Consent) vor Analyse erforderlich.  
- DPA abschließen, wenn als Auftragsverarbeiter tätig.  
- Data minimization: nur nötige Daten verarbeiten; Masking standardmäßig.  
- Betroffenenrechte: Auskunft, Löschung, Einschränkung — Prozesse dokumentieren.

Technische Maßnahmen:
- Encryption at rest (LUKS/GPG), signed manifests, audit logs, retention management.

Prüfung:
- DPA & Consent juristisch prüfen lassen.
